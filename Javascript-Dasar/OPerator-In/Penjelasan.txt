-In operator adlah operator yang bisa dilakukan untuk mengecek apakah sebuah property ada didalam object atau tidak
-Jika property tersebut ada di object, maka hasilnya true, sedangkan jika tidak, maka hasilnya false 
-Tidak hanya object, In juga bisa digunakan untuk mengecek index di array

# Perlu diingat
    - In property hanya akan mengecek apakah sebuah property atau index ada atau tidak 
    - Jadi walaupun nilai property atau index nya undefined atau null, maka tetap dianggap ada
